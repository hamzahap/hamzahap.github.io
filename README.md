# Hamzah Apps Site

This repository hosts the public developer website for Hamzah's mobile apps.

Planned live URLs:

- `https://hamzahap.github.io/`
- `https://hamzahap.github.io/app-ads.txt`
- `https://hamzahap.github.io/edgebloom/`
- `https://hamzahap.github.io/edgebloom/privacy-policy/`

The root `app-ads.txt` is shared across ad-supported apps that use the same
AdMob publisher account.
